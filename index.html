{Web: PyTe} by JoshAshby

PyTe for the modern web.

A senior capstone for high school to bring PyTe and all his awesomeness to the web for when PyTe isn't available, such as on mobile platforms.

Dependencies

Python 2.7 Couchdb Lighttp Less.js Coffeescript Others

Install

Placed somewhere in the code directories for the time being.
