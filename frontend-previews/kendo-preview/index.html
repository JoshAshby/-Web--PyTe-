PyTe: Web Edition

Version: .1a

PyTe stands for Python Text Editor PyTe was origonally written as a desktop application in Python using the QT3, and in later versions the QT4 Python bindings. As a result of the massive amount of technology coming out for the web, it's only fitting that a web version of PyTe should be available. This is the vist iteration of him, built with less.js, jquery, kendoUI, ace, keymaster.js, CoffeeScript and several other web technologies. This is a HTML5 web app, with a mobile view and a regular desktop view. It also takes advantage of new HTML5 technologies and CSS3.

Upload a file

Here you can choose a file on your harddrive which will be uploaded with HTML5's file API and the resulting text placed into the editor.

Welcome to {Web: PyTe}!

We're glad to have you here. This is the web version of the Python Text Editor by Josh Ashby. I'm built on HTML5, CSS3 and several javascript frameworks including less.css, kendoUI, jQuery, ace editor, and keyMaster.js.
All of these need pretty new browsers, so as a result, for your best experience please consider using Firefox or Google Chrome.

Okay, lets get you started here. I use localStorage to store your opened files, so all you have to do is go to the file menu and click open, or use the key sequence "ctrl a" which should open up the file selection window.
After this is done, I'll store the changes and everything for you in localStorage. That way, if you leave or close your browser, all the changes you've made will still be here. (Later on I may also store this in a server side database and provide a server project and file open function.)
